About PhishGuard

AI-Based Phishing Website Detection System · NPS Lab Project

🎯

Objective

Detect phishing websites in real-time using machine learning by analyzing 30 network and URL-level security features extracted live from any given URL.

📊

Dataset

Trained on the UCI Phishing Websites Dataset containing 11,055 website records. Each record has 30 features encoded as -1 (suspicious), 0 (neutral), or 1 (safe).

🌲

Model

Random Forest Classifier with 100 decision trees. Achieves ~96–97% accuracy with stratified train/test split. Uses ensemble voting to reduce false positives.

🌐

Network Analysis

Live HTTP requests, DNS/WHOIS queries, SSL certificate checks, redirect chain analysis, and anchor/form inspection are performed for every scanned URL.

30 Security Features

URL-Based (8)
  • IP address in URL
  • URL length
  • URL shortening service
  • @ symbol in URL
  • Double slash redirect
  • Prefix/suffix dash
  • Subdomain depth
  • HTTPS token in domain
Domain-Based (6)
  • SSL final state
  • Domain registration length
  • Favicon origin
  • Non-standard port
  • Domain age
  • DNS record exists
Page Content (9)
  • Request URL ratio
  • Anchor URL ratio
  • Tag link ratio
  • Server form handler
  • Email submission form
  • Abnormal URL
  • Iframe usage
  • Mouseover action
  • Right-click disabled
External (7)
  • Redirect count
  • Popup window
  • Web traffic rank
  • Page rank
  • Google index status
  • Inbound links count
  • Statistical report
