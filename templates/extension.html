How It Would Work

01

Page Load Intercept

Extension intercepts every navigation event using Chrome's webNavigation API before the page fully loads.

02

Feature Extraction

URL-based features (length, IP, symbols, subdomains) are extracted instantly in the browser without a server round-trip.

03

API Call to PhishGuard

The extension sends the URL to POST /api/scan and receives a JSON prediction with confidence score in milliseconds.

04

Warning or Pass

If phishing is detected, the extension injects a warning overlay. Safe sites get a green badge. User can override if needed.
